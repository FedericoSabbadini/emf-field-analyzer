================================================================================
2. THEORETICAL BACKGROUND
================================================================================

BIOT-SAVART LAW FOR INFINITE WIRE:

The magnetic field at distance r from an infinite straight wire is:

    B = (μ₀ · I) / (2π · r)

where:
    μ₀ = 4π × 10⁻⁷ H/m = 1.2566 × 10⁻⁶ H/m
    I = current in amperes
    r = perpendicular distance from wire in meters

DERIVATION:
Starting from the general Biot-Savart law:
    dB = (μ₀ · I) / (4π) · (dl × r̂) / r²

For an infinite wire along z, integrating over all z ∈ (-∞, +∞):
    B = (μ₀ · I) / (4π) · ∫_{-∞}^{+∞} dz / (r² + z²)^(3/2) · r
    B = (μ₀ · I) / (4π) · [z / (r² · √(r² + z²))]_{-∞}^{+∞} · r
    B = (μ₀ · I) / (4π) · (2/r)
    B = (μ₀ · I) / (2π · r)

FIELD DIRECTION:
The field is azimuthal (tangent to circles around the wire):
    B⃗ = B · φ̂

In Cartesian coordinates for wire at (xₛ, yₛ):
    Bₓ = -B · (y - yₛ) / r = -B · sin(φ)
    Bᵧ = +B · (x - xₛ) / r = +B · cos(φ)
    Bᵤ = 0

where φ = atan2(y - yₛ, x - xₛ) is the azimuthal angle.

================================================================================
5. KEY OBSERVATIONS
================================================================================

1. CIRCULAR SYMMETRY: The XY plane is perpendicular to the wire, so the 
   field exhibits perfect circular symmetry around the wire position.

2. 1/r DECAY: The field magnitude decreases inversely with distance.
   Doubling the distance halves the field strength.

3. SINGULARITY: The field diverges at r=0 (wire position). This is a 
   mathematical artifact of the thin wire approximation. Real wires have
   finite radius, limiting the maximum field.

4. QUASI-STATIC REGIME: At 50 Hz, wavelength λ = c/f ≈ 6000 km, so at 
   observation distances of tens of meters, retardation effects are 
   completely negligible.

5. REGULATORY COMPLIANCE: For I=100 A, the 3 μT quality objective 
   requires a clearance distance of approximately 7 meters.

using the superposition principle.
